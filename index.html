NWS Active Weather Alerts

This map displays all active weather alerts across the United States as of the current date. The data is sourced from the National Weather Service (NWS) API, specifically for active alerts, which provides information on all active warnings including event type, severity, urgency, and a link to more information. This map uses a simple color coding scheme to classify different levels of alert severity, with darker colors indicating more severe alerts (with the exception of purple, which is used for alerts of unknown severity).

  • Map created: Feb. 20th, 2026
  • Data source: weather.gov API (NWS), Active Alerts (specific URL)
